# Project Implementation PBO - Input Form PHP

Project ini adalah tugas kelompok untuk mata kuliah PBO yang mengimplementasikan Form Input menggunakan PHP dengan konsep OOP (Object Oriented Programming).

## Fitur
- Form Input (Firstname, Lastname, Phone, Address)
- Implementasi Class dan Object pada PHP
- Styling menggunakan CSS External
- Integrasi GitHub untuk kolaborasi tim

## Cara Menjalankan
1. Clone repository ini ke folder `htdocs`.
2. Jalankan Apache pada XAMPP.
3. Akses `localhost/project-pbo-implementation/index.php`.
